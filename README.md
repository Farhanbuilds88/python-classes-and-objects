Python OOP : Classes, Objects, Constructors, and Methods
Overview

This repository contains my learning and practice work related to Object-Oriented Programming (OOP) in Python.
The goal of this project is to understand the fundamental concepts of OOP such as classes, objects, constructors, functions, and methods inside classes.

These concepts are essential for writing organized, reusable, and scalable code in Python.

Concepts Covered
1. Classes

A class is a blueprint used to create objects.
It defines the structure and behavior that the objects created from it will have.

A class can contain:

Variables (attributes)

Functions (methods)

Example idea:
A Student class can store information like name, roll number, and marks.

2. Objects

An object is an instance of a class.
It represents a real-world entity created from the class blueprint.

Example:
If Student is a class, then:

Student 1 → object

Student 2 → object

Each object can have different values but follows the same structure defined in the class.

3. Constructor (__init__)

A constructor is a special method in Python that is automatically executed when an object is created.

In Python, the constructor is written as:

__init__()

It is mainly used to initialize object attributes.

Example purpose:

Assign name

Assign marks

Set initial values

4. Functions and Methods in Classes

Functions written inside a class are called methods.

Methods define the behavior of objects.

Examples of methods:

Display student information

Deposit money in a bank account

Calculate average marks

Methods always use the self parameter, which refers to the current object.

5. The self Keyword

The self keyword represents the current instance of the class.

It is used to:

Access instance variables

Call other methods inside the class

Example:

self.name
self.balance
