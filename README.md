# Python Object-Oriented Programming (OOP) Basics

## 📌 Overview
This repository contains my learning practice of **Object-Oriented Programming (OOP) in Python**.  
It focuses on understanding the fundamental concepts used to build structured and reusable programs using classes and objects.

---

## 📘 Topics Covered

- **Classes** – Understanding how classes act as blueprints for creating objects.  
- **Objects** – Creating instances of classes to represent real-world entities.  
- **Functions Inside Classes** – Learning how functions are defined within classes.  
- **Class Methods** – Understanding how methods operate on object data.  
- **Constructors** – Using constructors to initialize object attributes when objects are created.  
- **Instance Variables** – Storing object-specific data inside classes.  
- **`self` Keyword** – Understanding how the `self` keyword is used to refer to the current object and access its attributes and methods.

---

## 🎯 Purpose
The purpose of this repository is to practice and demonstrate the **basic concepts of Object-Oriented Programming in Python** and understand how these concepts help in writing **clean, modular, and reusable code**.
